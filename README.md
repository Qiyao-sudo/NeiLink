# NeiLink(清连)

![NeiLink](NeiLink.png)

## 📋 项目概述

NeiLink(清连) 是一款面向个人用户的跨平台局域网文件分享工具，专注于快速、安全地在本地网络中分享文件。解决传统传输方式（U盘拷贝、云盘上传下载、聊天软件发送）存在的速度慢、流量消耗大、文件大小限制、操作繁琐等痛点。

### 与传统传输方式对比

| 传输方式 | 速度 | 流量消耗 | 操作复杂度 | 文件大小限制 | 外网依赖 |
| --- | --- | --- | --- | --- | --- |
| **NeiLink** | 高速（局域网带宽） | 无 | 简单（一键分享、浏览器下载） | 无限制 | 无 |
| U盘拷贝 | 中等（取决于U盘读写速度） | 无 | 复杂（插拔U盘、查找文件） | 受U盘容量限制 | 无 |
| 云盘上传下载 | 低速（受外网限速） | 大 | 复杂（上传、下载两步操作） | 受云盘容量/会员限制 | 有 |
| 聊天软件发送 | 低速（受外网限速） | 大 | 中等（需添加好友/进入聊天） | 有明确大小限制 | 有 |

### 核心价值
- **高效传输**：基于局域网/热点传输，无外网依赖，速度取决于本地网络带宽
- **便捷操作**：一键发起分享，接收端仅需访问指定网址即可下载，无需安装额外软件
- **多端兼容**：覆盖Windows、macOS、Linux三大桌面系统，接收端支持各类设备浏览器访问
- **安全可控**：采用AES端到端加密，支持提取码验证、有效期与下载次数限制
- **灵活适配**：自动适配局域网与热点模式，断网重连后保留分享配置

## ✨ 核心功能

- 🚀 **文件/文件夹分享**：支持拖拽添加、右键分享，自动处理文件夹打包
- 🔒 **安全加密**：AES-256-CBC端到端加密，保护敏感文件
- 📱 **多设备支持**：PC、手机等多种设备通过浏览器访问
- 🎯 **文件码系统**：通过文件码快速定位和下载文件
- 🔑 **访问控制**：可设置提取码、有效期、最大下载次数
- 📊 **分享管理**：实时查看分享状态、下载统计，支持批量管理
- 🌐 **网络适配器选择**：支持选择特定网络适配器进行分享
- 🔥 **热点功能**：内置热点创建，应对AP隔离等特殊网络环境
- 📥 **断点续传**：支持大文件断点续传，提升下载体验
- 📝 **日志记录**：详细的分享与下载记录，便于追溯操作历史

## 🛠️ 技术架构

- **客户端**：Electron + Node.js，负责文件处理、网络适配、加密解密、日志记录
- **前端**：React + TypeScript + Ant Design，负责接收端的访问与下载交互
- **传输**：基于HTTP协议，结合AES端到端加密，保障传输安全
- **构建工具**：Webpack + TypeScript Compiler

## 📦 安装指南

### 从源码构建

1. **克隆仓库**
   ```bash
   git clone https://github.com/Qiyao-sudo/NeiLink.git
   cd NeiLink
   ```

2. **安装依赖**
   ```bash
   npm install
   ```

3. **构建项目**
   ```bash
   npm run build
   ```

4. **运行开发服务器**
   ```bash
   npm run dev
   ```

### 从发布包安装

访问 [Releases](https://github.com/Qiyao-sudo/NeiLink/releases) 页面下载对应平台的安装包。

## 📖 使用流程

### 分享文件
1. 点击「选择文件」按钮或拖拽文件/文件夹至软件界面
2. 配置分享选项（提取码、有效期、下载次数等）
3. 点击「开始分享」按钮
4. 复制生成的分享链接发送给接收方

### 接收文件
1. 打开分享链接
2. 如果设置了提取码，输入提取码
3. 点击「下载」按钮开始下载（支持断点续传）

### 管理分享
- 在「分享管理」页面查看所有活跃的分享任务
- 支持编辑分享配置、取消分享、查看下载统计

## 🌐 网络适配

- **局域网检测**：自动识别Wi-Fi、以太网等网络环境
- **热点模式**：无网络时自动启动热点，支持AP隔离场景
- **断网重连**：网络恢复后自动更新分享链接IP，保持分享配置

## ❓ 常见问题

### 无法访问分享链接
- 确保分享方和接收方在同一局域网内
- 检查防火墙设置，确保端口已开放
- 尝试切换网络适配器或开启热点

### 下载速度慢
- 确保网络连接稳定
- 避免同时进行大量网络活动
- 对于大文件，建议使用有线网络连接

### 分享链接失效
- 检查分享是否已过期
- 检查分享是否已被取消
- 检查网络连接是否正常
